# Audio assets

The site loads `/audio/ambient.mp3` when the user clicks the Klang button.
Drop a file at that exact path and it works.

Safe sources for a PD/CC-BY classical placeholder:
- IMSLP recordings explicitly tagged `Performance Restricted: Public Domain`
- Musopen "Public Domain" recordings (NOT their "Royalty Free Music" tier — that's CC-BY which requires attribution)
- Wikimedia Commons Featured Sounds (filter for PD)

The mic toggle works without any file.
